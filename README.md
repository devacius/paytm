# Paytm Clone
A simple paytm clone created using MERN stack with virtual money provided upon your signup . This project is created by me to learn about mongodb , creating backend, user authentication and understanding HTTP server requests. (It is not linked to you bank account No need to worry😅).
## Features:-
1. Using jsonwebtoken for user authentication.
2. Storing password through hashing .
3. You can send money to anyone registered in the database.
4. Secured using suitable applications.
## Installation:-
1. Clone the repository using git clone.
2. connect to your mongo database by providing your MONGO_URL.
3. run node index.js once in the backend folder.
4. run npm run dev once in the frontend folder.

 

       cd backend
        node index.js

In new terminal-

    cd frontend
    npm run dev
